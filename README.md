# Qualidade de Vida - Atividade Web

Este repositório contém uma página web informativa desenvolvida como parte da **10ª Atividade de Front-End**. O projeto aborda a importância do bem-estar físico e mental, aplicando conceitos fundamentais de estruturação e design web.

## Sobre o Projeto

A página apresenta um layout limpo e direto, contendo:
* **Conteúdo Informativo:** Textos que definem o conceito de "Qualidade de Vida" (saúde física, mental, lazer e equilíbrio).
* **Elementos Visuais:** Imagens ilustrativas de atividades físicas (corrida e alongamento) para reforçar o tema.
* **Design:** Uso de CSS para organização visual e tipografia.

## Tecnologias Utilizadas

* **HTML5:** Estrutura semântica (títulos `h1`, parágrafos `p`, imagens `img`).
* **CSS3:** Estilização do layout e posicionamento das imagens (arquivo `style.css`).
